# Kaarobar Web

Next.js marketing site + authenticated product app (`/app`).

```bash
cp .env.example .env.local
npm install
npm run dev
```

Requires `kaarobar-BE` at `NEXT_PUBLIC_API_URL` (default `http://localhost:4000/api/v1`).

Demo login after seeding the backend: `owner@kaarobar.local` / `password123`.
