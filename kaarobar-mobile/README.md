# Kaarobar Mobile

Expo (React Native) app for owners, managers, and employee self-service.

**Flow:** Landing → Sign in / Sign up → Dashboard (authenticated).

```bash
npm install
npm start
```

Set `EXPO_PUBLIC_API_URL` (default `http://localhost:4000/api/v1`).

Theme matches web/desktop Deep Sapphire (`#1d4ed8`).
